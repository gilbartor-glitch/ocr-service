Features

Everything you need to read any document

From raw scanning to AI explanation — one platform handles it all.

📖

Smart OCR

Advanced text recognition using Tesseract with custom Hebrew preprocessing. Handles skewed, low-res, and noisy scans.

🤖

AI correction

Claude AI automatically fixes garbled characters, fills in missing words, and reconstructs damaged text from context.

🧒

Explain like I'm 12

Complex legal, medical, or bureaucratic language translated into plain, simple sentences anyone can understand.

🌐

Instant translation

Translate extracted text into Hebrew, English, Arabic, Russian, French, Spanish, Amharic, and Tigrinya.

🔊

Text-to-speech

Listen to your document read aloud. Auto-detects language and uses the correct voice for Hebrew or English.

📦

Batch processing

Upload up to 20 images at once. Process entire document folders in a single API call or drag-and-drop session.

How it works

From image to understanding in seconds

1

Upload

Drop an image, paste a URL, or send via API. Supports JPEG, PNG, WEBP, TIFF, BMP.

2

Extract

Tesseract OCR reads the text. Hebrew and English are processed simultaneously.

3

Enhance

Claude AI fixes errors, simplifies language, and optionally translates — all automatically.

4

Use

Copy the text, listen via TTS, or receive clean JSON via the REST API.
